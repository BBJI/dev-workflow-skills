# 组件状态检查清单

在规格化组件（步骤五）时使用此清单，确保所有必要状态和行为已文档化。

## 每个交互组件的必要状态

| 状态 | 何时出现 | 需规格化什么 |
|------|---------|-------------|
| 默认 | 组件静止，无交互 | 视觉外观、使用的令牌 |
| 悬停 | 鼠标悬停在元素上 | 背景/边框/阴影变化、光标样式 |
| 聚焦 | 元素获得键盘焦点 | 聚焦环样式、偏移、颜色 |
| 激活 | 元素正在被按下/点击 | 按压的视觉反馈 |
| 禁用 | 元素不可交互 | 灰化、cursor-not-allowed、aria-disabled |
| 加载 | 等待数据/操作 | 骨架屏、旋转器或进度指示器 |
| 错误 | 校验失败或系统错误 | 错误样式、错误信息、aria-describedby |

## 组件特定状态

### 按钮
- [ ] 默认、悬停、聚焦、激活、禁用
- [ ] 加载状态（旋转器替代文本或与之并列）
- [ ] 图标+文本变体、纯图标变体、纯文本变体
- [ ] 主要、次要、幽灵、破坏性变体
- [ ] 尺寸变体：sm、md、lg

### 文本输入框
- [ ] 默认、聚焦（聚焦环）、禁用
- [ ] 占位符文本样式
- [ ] 已填充（有值）vs 空状态
- [ ] 错误状态：红色边框 + 下方错误信息
- [ ] 成功状态（可选）：绿色边框
- [ ] 带前缀/后缀图标或按钮
- [ ] 带字符计数
- [ ] 多行（文本域）：自动调整高度行为

### 选择器/下拉框
- [ ] 收起状态、展开状态
- [ ] 已选选项展示
- [ ] 选项列表：悬停、已选、禁用选项
- [ ] 搜索/过滤输入框（如可搜索）
- [ ] 空结果状态
- [ ] 加载选项状态
- [ ] 多选：已选项显示为标签

### 复选框/单选框
- [ ] 未选中、选中、半选（仅复选框）
- [ ] 悬停、聚焦、禁用
- [ ] 错误状态（校验失败）
- [ ] 组标签和描述

### 开关/切换
- [ ] 关闭、开启状态
- [ ] 悬停、聚焦、禁用
- [ ] 加载状态（如异步）
- [ ] 标签位置（左/右）

### 卡片
- [ ] 默认、悬停（如可点击）、选中
- [ ] 加载骨架屏
- [ ] 空状态
- [ ] 有/无图片
- [ ] 有/无操作区

### 模态框/对话框
- [ ] 遮罩/背景样式
- [ ] 进入动画、退出动画
- [ ] 关闭按钮、Escape键、点击外部关闭
- [ ] 滚动行为（body滚动锁定）
- [ ] 堆叠（多个模态框）
- [ ] 响应式：移动端全屏

### Toast/通知
- [ ] 进入动画、自动消失计时、退出动画
- [ ] 变体：成功、错误、警告、信息
- [ ] 有/无操作按钮
- [ ] 堆叠行为（多个Toast）
- [ ] 屏幕位置

### 表格/列表
- [ ] 加载状态（骨架行）
- [ ] 空状态（插图 + 消息 + 引导按钮）
- [ ] 排序状态（列标题指示器）
- [ ] 选中行
- [ ] 行悬停
- [ ] 行内编辑（如适用）
- [ ] 分页控件
- [ ] 响应式：移动端卡片布局或水平滚动

### 标签页
- [ ] 激活标签、非激活标签
- [ ] 非激活标签悬停
- [ ] 聚焦指示器
- [ ] 禁用标签（如适用）
- [ ] 溢出处理（标签过多）：滚动或下拉
- [ ] 面板切换：即时或动画

### 工具提示
- [ ] 显示延迟（通常300-500ms）
- [ ] 隐藏延迟
- [ ] 位置：上、右、下、左（自动调整以适应视口）
- [ ] 多行工具提示
- [ ] 箭头/指示器指向触发元素

## 响应式行为检查清单

对每个组件，文档化：

- [ ] `--bp-sm`（移动端）下的行为
- [ ] `--bp-md`（平板）下的行为
- [ ] `--bp-lg`（桌面端）下的行为
- [ ] 全宽 vs 固定宽行为
- [ ] 移动端触控目标尺寸（≥ 44x44px）
- [ ] 文本换行/截断规则
- [ ] 优先内容（各断点下什么隐藏/显示）
